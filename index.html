Real-Time Power Consumption & Cost Calculator

This project introduces a non-invasive, cost-effective device for monitoring household electricity consumption in real time. By optically reading the LED blink rate on a standard utility meter, the system calculates and displays power usage (in kWh) and its associated cost. The core objective is to provide homeowners with immediate, actionable data, fostering energy awareness and enabling significant cost savings.

  • High-Accuracy Blink Detection: Utilizes a sensitive Light Dependent Resistor (LDR) to monitor the energy meterâ€™s pulse LED. The system is calibrated based on the meter's standard (e.g., 1000 blinks/kWh), achieving >99% accuracy in detecting energy consumption pulses.
  • Real-Time Data Display: An integrated LED display provides instantaneous feedback on accumulated power consumption in kilowatt-hours (kWh) and the total estimated cost, updated in real time.
  • Adjustable Cost Tariff: Features a potentiometer that allows the user to manually set the per-unit electricity cost (e.g., from â‚¹3 to â‚¹15/unit), ensuring the financial calculations remain accurate even when utility rates change.
  • Informed Energy Savings: By providing transparent usage data, the device empowers users to identify high-consumption patterns and make informed decisions, potentially reducing monthly electricity bills by 10-20%.

Technical Specifications

  • Sensing Mechanism: High-sensitivity LDR sensor for non-invasive optical pulse detection.
  • Core Processor: ATmega328P microcontroller, serving as the central processing unit.
